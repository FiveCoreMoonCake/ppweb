# 🚀 Welcome to MDX

This page is rendered directly from a `.mdx` file inside the `app/article` folder. 

The metadata at the very top is **YAML frontmatter**. We've configured the `remark-frontmatter` and `remark-mdx-frontmatter` plugins so that this data is parsed and automatically exported. You can import this `.mdx` file elsewhere to read its metadata!

### Features Supported:

- Standard **Markdown** rendering.
- Native Next.js App Router routing.
- **Tailwind CSS** classes inside JSX.
